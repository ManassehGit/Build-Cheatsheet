HTML Tables

Table Reference #The Stylish CheatSheet


Tag Name Description
<table> Table Wrapper for the main Table
<thead> Table head Contains the header section of the table
<tbody> Table body Set of rows containing the various table data
<tr> Table row Container for the many table data
<td> Table data Table data is the basic unit of a table or basically a cell
<tfoot> Table footer Contains the footer section data for a table

Table Attributes

Attribute Name Description
colspan Column span Specifies the number of column spaces a table data would use
rowspan Row span Specifies the number of row spaces a table data would use
